# Unit Converter Tool

A simple and user-friendly Unit Converter application built with Python and Tkinter. This tool allows users to convert values between different units of length, weight, and temperature through an easy-to-use graphical interface.

## Features

- Convert Kilometers to Miles
- Convert Miles to Kilometers
- Convert Kilograms to Pounds
- Convert Pounds to Kilograms
- Convert Celsius to Fahrenheit
- Convert Fahrenheit to Celsius
- Convert Centimeters to Inches
- Convert Inches to Centimeters
- Convert Meters to Feet
- Convert Feet to Meters
- Simple graphical user interface (GUI)
- Input validation with error messages
- Instant conversion results

## Technologies Used

- Python 3
- Tkinter (GUI library)
